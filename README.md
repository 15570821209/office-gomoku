# 摸鱼五子棋 · 协作排期表

一个适合办公室搭子玩的 15 路五子棋。界面看起来像协作工作台，按 `Esc` 会立即切换成项目进度表。

## 启动

无需安装任何第三方库，只要电脑有 Python 3.10+：

```powershell
cd wuziqi
python server.py
```

然后浏览器打开终端里显示的地址：

- 自己玩：`http://127.0.0.1:8765`
- 和同一公司/家庭网络里的搭子玩：把终端显示的局域网地址发给对方，例如 `http://192.168.1.23:8765`

进入后选择“创建房间”，复制邀请链接给搭子。对方也可以输入六位房间码加入。

即使你从 `127.0.0.1` 打开，复制邀请时也会自动换成当前电脑的局域网 IP；请勿手动把 `127.0.0.1` 发给其他设备。

> 如果搭子打不开地址，请允许 Python 通过系统防火墙，并确认两台电脑在同一个局域网/VPN 中。

## 功能

- 本机双人和局域网房间对战
- 房间链接/六位房间码邀请
- 自动判定横、竖、斜线五连胜
- 撤回、重开、局数和比分统计
- 搭子暗号、音效开关、响应式棋盘
- `Esc` 老板键，切换到逼真的项目进度表
- 快捷键：`R` 重开、`U` 撤回

## 可选参数

```powershell
python server.py --port 9000
python server.py --host 127.0.0.1 --port 8765
```

房间数据只保存在运行服务的电脑内存中；服务关闭后自动清空，不会上传到外部网络。

## 免费部署到公网（Render）

项目已经包含 `Dockerfile` 和 `render.yaml`，可以作为 Render 免费 Web Service 部署。部署后双方访问同一个 HTTPS 地址，不再依赖局域网、IP、端口放行或个人热点。

1. 在 GitHub 新建一个仓库，把本目录内的文件上传到仓库根目录。
2. 登录 Render，选择 **New → Blueprint**，连接刚创建的 GitHub 仓库。
3. Render 会读取 `render.yaml`；确认实例类型为 **Free** 后创建服务。
4. 等部署状态变成 **Live**，打开 Render 提供的 `https://...onrender.com` 地址。

首次访问休眠的免费实例时可能需要等待约一分钟。免费实例空闲 15 分钟后会休眠，服务重启或休眠会清空内存中的旧房间；重新创建房间即可，不影响正在持续访问的对局。

Render 部署时会自动把平台提供的 `PORT` 环境变量交给服务；本地启动仍默认使用 `8765`。
